## Description

This is a web page which is basically a web page template that is meant for some customer to see. Nothing too complex and full of lorem ipsum.

## Tools

- Next.js with TypeScript.
- Tailwind CSS - to stylizing.
  - Tailwind-merge, CVA and clsx - to help with dynamic style without conflict.
- Swiper - to help with slider.
- Frame-motion - Facilitate the implementation of animations.
- Hamburguer-React
- React icons
